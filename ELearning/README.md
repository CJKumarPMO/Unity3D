# ELearning

This is the ELearning microservices platform.

## How to run (once Docker is installed)

1. Open the folder in VS Code: `C:\Users\KumaravelChinappanJa\Unity3D\ELearning`
2. Run:
   ```bash
   docker-compose up --build
   ```

## Services

- API Gateway: http://localhost:3000
- User Service: http://localhost:3001
- Course Service: http://localhost:3002
- Enrollment Service: http://localhost:3003
- Frontend: http://localhost:3004
